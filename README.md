# Magic-Thrones
Group project made with HTML5 Canvas

### Controls
```
W, S, A, D or ArrowKeys to move
[Spacebar]              to Jump
[Shift]                 to shoot

```

Try [Magic Thrones](https://shoppinglist-voster.herokuapp.com/MagicThrones/ "A HTML5 canvas & JS game")!
